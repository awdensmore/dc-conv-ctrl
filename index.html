Project Notes

This is a general list of notes of important tips I've learned along the way.

  • PWM using DMA - This is useful to rapidly change the PWM duty cycle without accessing the CPU. Some important tips when using CubeMX to generate the setup code: ''' C hdma_tim1_ch1.Instance = DMA1_Channel2; hdma_tim1_ch1.Init.Direction = DMA_MEMORY_TO_PERIPH; hdma_tim1_ch1.Init.PeriphInc = DMA_PINC_DISABLE; hdma_tim1_ch1.Init.MemInc = DMA_MINC_ENABLE; hdma_tim1_ch1.Init.PeriphDataAlignment = DMA_PDATAALIGN_WORD; hdma_tim1_ch1.Init.MemDataAlignment = DMA_MDATAALIGN_WORD; hdma_tim1_ch1.Init.Mode = DMA_CIRCULAR; hdma_tim1_ch1.Init.Priority = DMA_PRIORITY_HIGH; HAL_DMA_Init(&hdma_tim1_ch1);
